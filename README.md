# work-scripts
Scripts used at work

## Repository contains:
```
.
├── Log_Filter
│   └── Python script for parsing log files based on specified keywords
├── PDF_Converter
│   └── Python script for converting PDF files to PDF/A1 format
├── get_installed_programs_list
│   └── Powershell script for gathering information about installed programs on computer
├── get_windows_logs
│   └── Powershell script for collecting specified system log files from computers in local network
└── remove_inactive_ad_user_profiles
    └── Powershell script for removing local profiles of disabled domain users
```

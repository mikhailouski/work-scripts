Этот powershell-скрипт позволяет автоматически конвертировать несколько PDF-файлов в формат PDF/A-1 (архивный стандарт) с использованием Ghostscript


   Предварительная настройка

1. Необходимо скачать и установить Ghostscript (установочный файл gs10051w64.exe для win10x64 находится в текущей папке)
Проверить установку:
	в командной строке
	gswin64c --version
2. Подготовить рабочую директорию:
	D:\PDF_Converter\
	├── input\    (исходные PDF)
	├── output\   (готовые PDF/A)
	└── convert.ps1
3. Поместить исходные PDF в папку input.


   Запуск скрипта

Запустить PowerShell от имени администратора и выполнить:
	Set-ExecutionPolicy RemoteSigned -Scope CurrentUser  	# Разрешить запуск скриптов
	.\convert.ps1                                       	# Запуск конвертации

В результате работы скрипта готовые файлы появятся в папке .\output c суффиксом _PDFA
Результаты и ошибки сохраняются в файл conversion_log.txt
